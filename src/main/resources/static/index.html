安全风险提示

本系统 故意包含 以下安全漏洞，仅供安全教学演示使用：

  • Swagger Swagger UI 未授权访问，API 接口和数据模型完全暴露
  • .git .git 目录可通过 Web 访问，源码和提交历史泄露
  • 目录遍历 文件下载接口未做路径校验，可读取任意文件
  • 敏感配置 数据库中包含硬编码密码和密钥
  • 信息泄露 用户接口返回明文密码等敏感字段
安全漏洞演示系统 · 请勿在生产环境中使用
