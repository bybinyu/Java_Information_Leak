===========================================
 文件管理平台 —— 项目概述文档
===========================================

一、项目背景
本文件管理平台服务于企业内部文档流转需求，支持文件
上传、存储、在线预览和下载等功能。

二、技术架构
- 后端: Spring Boot + Java 17
- 前端: 静态页面 + RESTful API
- 存储: 本地文件系统
- 文档: OpenAPI / Swagger
